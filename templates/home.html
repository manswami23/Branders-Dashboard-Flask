Who We Are

We are Branders. Branders is an analytic dashboard equipped with features that help brand managers expand their social media presence. With full time support and relevant data, users are able to learn great insights about their platforms audience.

Why Us

Brand Tracking + Social Media Posting + Current Events

User Friendly

Intuitive enough for one user or add colleagues

With a single click of a button, users have the ability to access a wide range of features. Our dashboard allows for team members to not only collaborate with one another on potential posts, but also to view relevant data about their audience.

Analytics

Import data for R analysis

After gathering data from their social media profiles, users have the ability to create their own charts and graphs. With this feature members can create a predictive analysis of trends surrounding their business. Additionally, users can keep track of key statistics such as impressions, mentions, and follower engagement.

Social Media

Unlock real time posting following current events

In the News and Media section, users are able to draft and send messages directly from their social media platform. The section is also equipped with a set of popular news articles that users can view when drafting messages. While the articles get updated regularly, users can use the search engine to find more information.

Open Sourced Tools

Free tools that allow you to add more custom functionalities

By using Flask development environment, we are able to take advantage of the full power of Python. Python is the largest developer community so there will be plenty of help debugging problems. It also boasts tons of libraries and APIs - this includes the Twitter and Facebook API. There is no limit on the custom features that could be added to this tool.
